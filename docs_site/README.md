# Agentic Executables Docs Site

VitePress-based docs product for human and AI-agent onboarding.

## Commands

```bash
npm install
npm run dev
npm run quality
```

## Build outputs

- Static docs site from `docs/`
- Machine-readable docs:
  - `/llms.txt`
  - `/llms-full.txt`

`llms` files are generated by `scripts/generate-llms.mjs` during `npm run build`.
